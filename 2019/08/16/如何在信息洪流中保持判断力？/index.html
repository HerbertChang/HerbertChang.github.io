如何在信息洪流中保持判断力？

发表于 2019-08-16 | 分类于 阅读与思考 | |

尤瓦尔·赫拉利在《人类简史》大卖之后，继续出版了《今日简史》和《未来简史》，长期占据国内许多书店热卖图书区。出乎很多人意料，这位纵论古今的学术新秀和「网红」作家日常不使用智能手机，每年都要抽两个月闭关静修一阵。与傅盛对谈中赫拉利说，「现在这个世界发生的事情，就像洪流一下过来……信息像潮水一样不断涌来，主要的问题就是能不能从信息里判断重要的和不重要的（内容）。你必须要可以集中注意力来找到一种信息上面的平衡」。

身处移动互联网时代的当代人既要面对海量信息每天涌入带来的焦虑与信息过载，同时要处理推荐算法导致的信息同温层（又称信息泡泡，指算法不断给推荐读者喜欢看的内容）问题。通过合理优化知识储备和媒介素养，有助于在信息洪流中保持头脑清醒，培养独立判断力。


常识感与逻辑的养成


任何领域的知识都是有一定门槛的。生活中大家都知道听出租车司机跟你谈天下大事不可信，为什么这样的内容搬到了微信号、今日头条上你就信了？为什么不自己去看刘强东案的149页文件而是去转发标题带有噱头、内容上歪曲事实的文章？许多所谓的「自媒体」并不真的懂行、只是想吸引眼球，靠拼凑一些文本片段制造看起来很有知识有观点的假象。要识破那些假装有知识其实是草包的文章，需要一些常识感和逻辑思考力（大部分观点站不住脚的文章逻辑上就能找出很多硬伤）。

在常识感的养成方面，推荐两套书：一是名家通识讲座书系（也被称为北大「十五讲」系列），这套书是北京大学发起、全国十多所重点大学和一些科研单位协作编写的一套大型多学科普及读物，涵盖文、史、哲、艺术、社会科学、自然科学等多个学科领域，可读性相当好。其二是牛津通识读本系列，每一册都不厚但对于相应领域都能以深入浅出的方式介绍，非常适合作为地铁读物。


逻辑思考力的养成方面，推荐欧文·柯匹的《逻辑学导论》。



媒介素养与媒体食谱


按照维基百科的解释，「媒介素养（Media Literacy）是指在各种环境中以适合自己的目的和需求的方式取用（access）、分析（analyze）、评估（evaluate）及制造（create）媒体信息的能力，又称媒介素养。“媒体素养教育”或“媒体教育”有别于培养媒体从业人员的媒体专业教育，教育对象主体是全体公民。教育目标为使全民具备自主思辨能力与产生信息的能力，从而以批判性角度去解读各种媒体信息、假新闻。」

在当下社交媒体中谣言漫天飞、假新闻比辟谣帖跑得快的情况下，提高自我媒介素养、避免虚假信息对大脑的侵蚀尤为重要。方可成老师提出「媒体食谱」的概念，「有句话说，吃什么，你就是什么。其实，这句话从物质食粮扩展到精神食粮同样成立——读什么，你就是什么」。在Bilibili网站上有方可成在TEDx演讲的视频，题为《如何优化你的媒体食谱》。还要特别推荐方可成老师的媒体食谱系列文章。这个系列文章中有介绍如何将邮箱调教成为最好的新闻阅读器、如何读外媒、对碎片化认知的处理等内容，值得认真研读并付诸行动。
